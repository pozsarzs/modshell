The source package contains files for building the following types of packages:

DOS

  • EXE binary package
  • ZIP binary package
  • ZIP binary package for FreeDOS

FreeBSD

  • BIN installer
  • ZIP portable package
  • TXZ binary package

Linux

  • BIN installer
  • ZIP portable package
  • DEB binary package
  • RPM binary package
  • TXZ binary package

Windows

  • ZIP portable package
  • EXE installer
  • MSI package
