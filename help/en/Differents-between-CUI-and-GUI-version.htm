The ModShell program has a command-line and is also suitable for running pre-created scripts independently as a command interpreter. The XModShell program has graphical interface (GUI), which helps to perform several operations from menu. Command line input remained available to them, but not always the same as the original solution. Commands are issued from a menu, but interaction appears on the console, except for the request to overwrite an existing file. The assignment of hotkeys remains unchanged. It is important to know that it can run loaded script but can't run independently as a command interpreter.
